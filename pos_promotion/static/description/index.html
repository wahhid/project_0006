Bruce's Services.

1. The first: you need define your promotions. Go to Menu Point of sale > Promotions > Promotions

2. Create new promotions
- Name: Promotion name
- Sequence: Sequence of Promotion
- Type: 3 options
+ Total Order: when pos order great or equal min total config, system will discount
+ Product by Product: for sell Product A, gift product B, yo need define more rule on bottom pages (Rule Line Products).
+ Product by Category: will be discount by your category chosen.
- Method: + Percent : use for percent % under total pos order + Discount Money: discount money under total pos order - Discount with: This is product template have type is "Service", will be display screen pos order - Min Order: this is condition Min of Order for promotions run. - Rule Line Products: when you chosen type is Product by Product, you need define product sell and product use for gift - Categories: when you chosen type is "Product by Category", you need define Category will be apply rule

3. When step (2) is Final, you login with admin and go to Menu Config > Point Of Sale > Click any Record you want apply Promotions - Scroll down page ,you can see Text "Promotions Apply" - Edit record and add promotions you need run.

4. Go to Dashboard and Go to Screen POS and check it. - When POS Order Done, click button Validate Promotions
